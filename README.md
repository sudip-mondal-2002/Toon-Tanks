# Toon-Tanks

## Destroy all the enemy tanks

## Controlls
| Key | Move |
|---|----|
| W | Move Forward |
| S | Move Backward |
| A | Rotate Left |
| D | Rotate Right |
| Left Mouse Click | Fire |
| Mouse Cursor | Aim |
